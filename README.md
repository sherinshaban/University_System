A comprehensive desktop application designed to streamline university administration tasks. This system manages students, teachers, and financial records efficiently using a user-friendly interface.

##  Features
The system includes several modules to handle different aspects of university management:
* **Secure Login:** Authentication system for admin access.
* **Student Management:** Add, update, view, and remove student records.
* **Teacher Management:** Manage faculty details, including hiring and payroll.
* **Fee Submission:** A module to handle student tuition fees and financial records.
* **Examination & Reports:** Generate report cards and manage academic performance.
* **Utility Tools:** Includes tools like Calculator, Notepad, and Web Browser integration.

##  Tech Stack
* **Language:** Java (Core Java & Swing/AWT).
* **IDE:** NetBeans IDE.
* **Database:** SQL Server (university_system)
* **Tools:** JDK 8 or higher.
